          광고 영역

          wsQf-PDF는 어떤 서비스인가요?

          wsQf-PDF는 설치나 회원가입 없이 브라우저에서 바로 쓰는 무료 PDF 정리 도구입니다. 여러 PDF를 하나로 합치고, 용량을 줄이고, 페이지 번호를 넣고, 도장이나 서명을 찍고, 영수증 사진을 A4 용지에 자동 정렬해 합계까지 계산한 PDF로 만들 수 있습니다.

          모든 처리는 사용자의 브라우저 안에서만 이루어집니다. 업로드한 파일과 사진은 서버로 전송되지 않으며, 탭을 닫으면 브라우저 메모리에서 사라집니다. 계약서, 견적서, 영수증처럼 민감한 문서를 다룰 때에도 외부 유출 걱정 없이 사용할 수 있습니다.

          도구별 사용 방법

          PDF 합치기

          파일 보관함에 PDF를 여러 개 올린 뒤 합치기 탭에서 원하는 파일을 체크하고 ↑↓ 버튼으로 순서를 정하면 하나의 PDF로 병합됩니다. 견적서와 계약서를 한 파일로 묶어 보내거나, 장별로 스캔한 문서를 하나로 정리할 때 유용합니다.

          PDF 압축

          스캔 문서나 사진이 많은 PDF의 용량을 크게 줄입니다. 압축 강도를 3단계에서 고를 수 있으며, 압축 후에는 문서 안 글자를 선택·검색할 수 없게 되는 점을 화면에서 미리 안내합니다. 이메일 첨부 용량 제한에 걸릴 때 특히 유용합니다.

          페이지 번호 넣기

          위치 6곳(상·하단 × 좌·중·우), 형식 3종, 시작 번호를 골라 모든 페이지에 번호를 자동 삽입합니다. 제안서나 보고서 제출 전에 페이지 번호를 요구받았을 때 다른 프로그램 없이 바로 해결됩니다.

          도장 · 서명 넣기

          도장 이미지를 올리거나 화면에 직접 서명을 그린 뒤, 미리보기 페이지를 클릭해 원하는 위치에 찍습니다. 드래그로 위치를 옮기고 크기를 조절할 수 있어 계약서 서명, 확인 도장 날인을 종이 출력 없이 처리할 수 있습니다.

          영수증 정리

          영수증 사진을 올리고 금액을 입력하면 A4 용지에 2·4·6장씩 자동 정렬되고 마지막 장에 합계 금액이 표시됩니다. 경비 정산, 지출 증빙 제출용 문서를 몇 분 만에 만들 수 있습니다.

          자주 묻는 질문

          무료인가요? 회원가입이 필요한가요?

          모든 기능이 무료이며 회원가입, 로그인, 결제가 전혀 없습니다.

          업로드한 파일이 서버에 저장되나요?

          아니요. 파일은 서버로 전송되지 않고 사용자의 브라우저 안에서만 처리됩니다. 탭을 닫거나 새로고침하면 작업 중이던 파일은 사라집니다.

          모바일에서도 사용할 수 있나요?

          네. 스마트폰과 태블릿 브라우저에서도 모든 도구를 사용할 수 있도록 화면이 자동으로 조정됩니다.

          암호가 걸린 PDF도 처리할 수 있나요?

          암호화된 PDF는 열 수 없습니다. 암호를 해제한 뒤 업로드해 주세요.

          압축하면 화질이 많이 나빠지나요?

          압축 강도를 약하게·보통·강하게 중에서 고를 수 있습니다. 다만 압축 방식의 특성상 압축 후에는 문서 안 글자를 선택하거나 검색할 수 없게 됩니다.

          영수증 금액은 자동으로 인식되나요?

          자동 인식(OCR)은 사용하지 않습니다. 금액을 직접 입력하면 합계가 자동으로 계산됩니다. 잘못 인식될 걱정 없이 정확한 금액으로 정리할 수 있습니다.

          한글이 깨지지 않나요?

          네. 페이지 번호, 영수증 정리 등 모든 결과 PDF에서 한글이 정상적으로 표시됩니다.

          개인정보 보호

          • 업로드한 PDF·이미지는 서버로 전송되지 않으며 브라우저 안에서만 처리됩니다.
          • 회원가입이 없으므로 이름, 이메일 등 개인정보를 수집하지 않습니다.
          • 다크 모드 설정만 브라우저(localStorage)에 저장되며 외부로 전송되지 않습니다.
          • 브라우저를 닫으면 작업 중이던 파일은 모두 사라집니다.
          • 민감한 개인정보가 포함된 문서는 결과 PDF를 공유하기 전에 직접 확인해 주세요.
          • 광고 게재 시 광고 사업자(예: Google AdSense)가 쿠키를 사용할 수 있으며, 이는 광고 적용 시점에 이 안내에 구체적으로 반영됩니다.
